Operating systems

  • I primarily use Gentoo Linux. To quote, “Because I hate myself.” Gentoo linux is a source-based distro meaning the source code is compiled locally on thy machine. Another aspect, USE flags, which can change which files, dependencies, libraries and such are included during installation.

  • I use Arch Linux btw. It's very minimal and gets the job done. Arch is bleeding edge meaning it offers the latest stable versions of most software.

  • Debian is stable, reliable, beautiful, and a great starting distro. Unlike Arch, Debian is not a cutting edge distro and will likely get major updates once every few years. This is both a drawback and a feature. You're risking latest updates of your favourite softwares for boasting a stable, NASA-level workstation. I use Debian when I don't feel like clinically insane.

Web browsers

  • Linux

    • Firefox is a decent browser as it is, but it comes with telemetry and Google services. I use a custom user.js file of my own based on Betterfox. At heart, Librewolf is a custom version of Firefox, focused on privacy, security, and freedom. It's quite privacy-centric out of the box if you don't want to bother customization.

    • For Chromium experience, Ungoogled Chromium removes all Google-specific web services. It's bare metal which can be a drawback to many. For example there is no extension menu or search engine by default.
      Brave is another good chromium fork which replaces google search engine with Brave search engine. They shill NFTs and brave wallet out of the box which I could not care less of so I turn them off. Otherwise it's a fairly decent browser.

  • Android

    • Firefox is the official Firefox for mobile. Fennec F-Droid is a better Firefox for mobile without proprietary code, telemetry, and ads. Complements extensions. I personally prefer Fennec over official app.

Mobile applications

also read: Android's Dark Age

  • Droidify, the better F-Droid, material you, fast.

  • Aurora Store is a lightweight Play Store alternative. Lets you use Play Store without login by anonymous session. Spoof device location to access geo-locked apps. Tells you whether Play Service is required or if there are trackers in the app.

  • Gramophone is a slick and minimal offline music player that can play .mp3, .flac, .ogg, .opus, .m4a, .mka and .wav files.

  • Simple Song Collector is a simple app that allows you to download audio from YouTube/YouTube Music. If you care about audio cover: YT Music covers are awfully blurry and YT video covers are clear.

  • ReadEra is a beautiful, all-in-one, multipurpose document and ebook reader.
